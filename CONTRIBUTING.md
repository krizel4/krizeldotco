# Contributing to Krizel

Thank you for your interest in contributing to Krizel! We welcome contributions that help make fitness resources more accessible.

## How to Contribute

### Reporting Issues

- Check if the issue already exists
- Use a clear and descriptive title
- Provide detailed steps to reproduce
- Include screenshots if applicable

### Suggesting Enhancements

- Use a clear and descriptive title
- Provide a detailed description of the suggested enhancement
- Explain why this enhancement would be useful

### Pull Requests

1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Run `npm run lint` to check for errors
5. Test your changes thoroughly
6. Commit your changes (`git commit -m 'Add some amazing feature'`)
7. Push to the branch (`git push origin feature/amazing-feature`)
8. Open a Pull Request

## Code Style

- Use TypeScript for type safety
- Follow the existing code structure
- Use Tailwind CSS for styling
- Write clear, descriptive commit messages
- Add comments for complex logic

## Content Guidelines

When adding blog posts or resources:

- Ensure accuracy of mental health information
- Cite sources when appropriate
- Use inclusive and sensitive language
- Avoid stigmatizing terminology
- Include content warnings when necessary

## Resources for Contributors

### Mental Health Content Best Practices

- Use person-first language ("person with depression" not "depressed person")
- Avoid graphic descriptions of self-harm or suicide methods
- Include crisis resources prominently
- Focus on hope and recovery

### Crisis Resources to Include

Always include these crisis resources where appropriate:

- **National Suicide Prevention Lifeline:** 988
- **Crisis Text Line:** Text HOME to 741741
- **SAMHSA National Helpline:** 1-800-662-4357

## Community Guidelines

- Be respectful and inclusive
- Focus on constructive feedback
- Respect diverse perspectives and experiences
- Maintain confidentiality and privacy
- Follow our Code of Conduct
